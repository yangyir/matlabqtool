Readme of SigMining

1. 定义

各变量间关系：ind（实数）=》sig（可数集）=》sigTag（待测的集合集合名）

指标indVal:
实数序列，用作生成各种交易信号。序列长度必须和sig实例中的价格序列长度相同，其它无要求。例如,[4.2,3.1,-1,5.3, ... ]

指标名indName: 	记录指标生成方法(例如macd)、参数等可还原的必要信息。

信号sigVal:
在[-1,1]之间的可数序列，每个非0值代表一次交易，大于（小于）0的值意为买入（卖出）。信号序列的长度需要和sig实例中的价格序列长度相同。例如，[-1,0,0,0,1,0,0,-1,...]

信号名sigName:
记录信号产生方法(例如@sign)、参数等可从indVal转成sig的必要信息。

信号标签sigTag: 列表(默认去除'0'),是被测的信号标签。例如：{'-1','1'}

进入entry: 开仓列表。长度不限。

退出exit: 关仓列表，和entry对应。长度不限。
